Forgotten Forest

Month-long Solo Project

About

Made in one month, Forgotten Forest is a solo project top down action game prototype featuring projectile centered combat, several enemy types, and two boss battles

Development Info

  • Passion Project
  • 1 month development cycle (10/11/2018 - 11/11/2018)
  • 1 developer, 2 musicians
  • Game Maker Studio 2

Contributions

  • Programmer
  • Combat Designer
  • Level Designer
  • UI/UX Designer

Post-Mortem

About the Project

Originally, I started work on Forgotten Forest during WolverineSoft's 2018 Pumpkin Game Jam, though it was not an actual submission as I was unable to attend. After getting permission, I presented it at the jam's Post-Mortem, and overall it seemed that people enjoyed it and were impressed. They gave me rather good feedback about ways to improve the combat, and following that, I worked more on the game throughout the rest of October and into November, before setting it aside.


What Went Right

  • Learned a lot about Game Maker
  • Learned a lot about pixel art
  • Designed a semi-interesting combat loop
  • Finished a game




What Could Be Improved

  • Knockback is a bit extreme
  • Boss fights are either not challenging enough, or impossible
  • Enemies could use some telegraphs
  • Controls could use some work
  • Balance item drops and abilities




Lessons Learned

  • Spend more time prototyping before working on finished assets
  • Get feedback early and often
  • Have more non-gamers playtest
  • Make code more abstract and modular to allow for easier expansion later
